# Student-Performance-Prediction-System
Project Overview :

 This project is a Machine Learning Web Application that predicts a student's overall performance based on academic, demographic, and behavioral factors.

It uses:

 *Python
 
 *Machine Learning (Random Forest)
 
 *Streamlit for the web interface
 
 The system helps identify student risk levels, expected grades, and provides a visual dashboard for analysis.

 Features

* Predict student performance score
* Automatic grade classification
* Risk level indicator (Low / Medium / High)
* Interactive graph dashboard
* Easy-to-use Streamlit interface
* Works with real dataset
